Mission

“If you can’t explain it to a 6-year-old, you don’t understand it yourself.”

-Albert Einstein



Holexp brings innovation in both the form and the content of teaching programming and electronics. Holexp abstracts all software syntax so that learners use plain text in coding and build up their confidence, engagement and skills through a unique platform. Holexp is powered by AI and uses NLP (in Arabic or English) as input to create functional automation projects using its open source hardware kit.

Vision

الرؤية

With Holexp, coding has never been easier! Holexp simplifies what is considered difficult in coding and hardware design. Programming and electronics is now for all: youth and anyone who have missed out on programming or electronics. Different from existing tools, Holexp engages minds and sets them on a path of lifelong learning. Write it in your own language and Holexp will create a python code that is compiled and executable by Holexp open source hardware kit.

مع هولكسبي، لم تكن البرمجة أسهل من أي وقت مضى! يبسط هولكسبي ما يعتبر صعبًا في الترميز وتصميم الأجهزة. البرمجة والإلكترونيات متاحة الآن للجميع: الشباب وأي شخص فاته البرمجة أو الإلكترونيات. تختلف عن الأدوات الحالية ، تعمل هولكسبي العقول وتضعها على مسار التعلم مدى الحياة. اكتبه بلغتك الخاصة وسيقوم هولكسبي بإنشاء كود بيثون الذي يتم تجميعه وتنفيذه بواسطة مجموعة أدوات هولكسبي مفتوحة المصدر.

About Us

التعرف علينا

Holexp

Holexp offers a rich hardware kit that includes a Bluetooth module, 4 different Microcontrollers, 7 Digital Inputs (push button, infrared receiver, tilt switch, infrared presence sensor, ultrasonic distance sensor, compass, real time clock), 9 Analog Inputs: (temperature, humidity, rain, pressure and altitude sensors, soil humidity sensor, microphone, light-dependent resistor, potentiometer), 7 Outputs ( led, multicolored led, buzzer, 2x16 characters LCD, DC motor, stepper motor, servo motor).
